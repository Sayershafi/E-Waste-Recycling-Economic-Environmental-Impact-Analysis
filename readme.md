##  Project Objective

This project explores the **economic and environmental impact of e-waste recycling in the United States**.

It combines national and state-level data to:

- Analyze recycling growth trends  
- Measure revenue generation  
- Evaluate CO‚ÇÇ emission reductions  
- Assess metal recovery performance  
- Identify top-performing states  
- Examine economic‚Äìenvironmental correlation  

The goal is to demonstrate how sustainability and profitability can align through data-driven decision making.
